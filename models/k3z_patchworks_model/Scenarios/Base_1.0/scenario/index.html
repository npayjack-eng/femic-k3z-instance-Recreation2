Scenario data files, Base_1.0 scenario

The files in this folder are data files that describe the state of the Patchworks simulation model at the time that the reports were saved. These reports are in CSV format, and can be inspected using database or spreadsheet software such as Access or Excel. You can load these files
  • into Patchworks to restore the state of the simulation,
  • or into other modelling software for extended analysis.

  • Event timeline
  • Scheduled treatments
