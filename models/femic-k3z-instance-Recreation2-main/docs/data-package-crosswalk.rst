Data Package Crosswalk
======================

This crosswalk maps BC small-unit timber supply data-package sections used in
the TFL26, CFA, and FNWL reports to standalone K3Z documentation pages.

   * - Non-Timber Assumptions
     - ``assumptions-registry.rst``
   * - Harvesting Assumptions
     - ``assumptions-registry.rst``
   * - Growth and Yield Assumptions
     - ``assumptions-registry.rst``
   * - Natural Disturbance Assumptions

   * - Modeling Assumptions

   * - Discussion

       ``land-base-and-netdown.rst`` and ``base-case-analysis.rst``
   * - References
     - ``base-case-analysis.rst`` and ``assumptions-registry.rst``
